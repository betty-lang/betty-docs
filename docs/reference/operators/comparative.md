# Comparative Operators

| Operator | Description               | Example         | Result  |
|----------|---------------------------|----------------|---------|
| `==`     | Equal to                  | `5 == 5`       | `true`  |
| `!=`     | Not equal to              | `5 != 3`       | `true`  |
| `>`      | Greater than              | `7 > 4`        | `true`  |
| `<`      | Less than                 | `3 < 8`        | `true`  |
| `>=`     | Greater than or equal to  | `6 >= 6`       | `true`  |
| `<=`     | Less than or equal to     | `2 <= 5`       | `true`  |
